# ImageProcessorJS

ImageProcessorJS is an image processing simulator where all image processing algorithms are implemented using pure JavaScript.

## [Webpage](https://erendn.github.io/image-processor-js/)

## Features
* Save image (Ctrl + S or &#8984; + S)
* Bit depth
* Dithering
    * Floyd-Steinberg
* Filters
    * Flip
    * Grayscale
        * Desaturation
        * Average
        * Luminance
        * Decomposition
        * Single color
    * Monochrome
    * Dichrome
    * Glass
    * Blur
        * Motion blur
        * Box blur
        * Tent blur
    * Posterize
    * Solarize
    * Edge Detection
